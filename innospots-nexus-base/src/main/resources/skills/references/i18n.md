# I18n (`com.innospots.nexus.base.i18n`)

## I18n

**Type:** `@interface`

Jackson annotation that marks a field for automatic i18n translation during serialization. The annotated field's value is passed through `I18nConverter.translate(Object)` before being written to JSON.

### Inner Class
- **I18nFieldSerializer** — extends `JsonSerializer<Object>`. Calls `I18nConverter.translate(value)` and writes the result via `JsonGenerator.writeObject()`.

---

## I18nConverter

**Type:** final class

Core i18n translation engine. Supports two translation modes: (1) I18n key resolution — strings wrapped in `${key}` are resolved via an `I18nMessageResolver`, and (2) `I18nObject` translation — `I18nObject` instances return the value for the current thread-local `Locale`. The current locale is stored in a `ThreadLocal`.

### Static Constants
- `Set<String> ZH_LOCALES` — set of Chinese locale identifiers (`zh`, `zh-CN`, `zh_CN`, `zh-Hans`, `zh_Hans`, `zh-TW`, `zh_TW`, `zh-HK`, `zh_HK`)
- `Set<String> EN_LOCALES` — set of English locale identifiers (`en`, `en-US`, `en_US`, `en-GB`, `en_GB`, `en-CA`, `en_CA`, `en-AU`, `en_AU`)

### Static Method
- **Signature:** `setLocale(Locale locale) → void`
- **Description:** Sets the current thread's locale for i18n resolution. Passing null clears the thread-local (fallback to system default).
- **Parameters:** `locale` — the locale, or null to clear

### Static Method
- **Signature:** `locale() → Locale`
- **Description:** Returns the current thread's locale, falling back to the JVM default.
- **Parameters:** none
- **Returns:** `Locale`

### Static Method
- **Signature:** `setMessageResolver(I18nMessageResolver resolver) → void`
- **Description:** Registers a global message resolver. Stored in an `AtomicReference`, so it can be set once at application startup.
- **Parameters:** `resolver` — the message resolver implementation

### Static Method
- **Signature:** `ignoreI18n() → void`
- **Description:** Globally disables i18n translation for the current thread.

### Static Method
- **Signature:** `shouldIgnoreI18n() → boolean`
- **Description:** Returns whether i18n translation is globally disabled for this thread.
- **Returns:** `boolean`

### Static Method
- **Signature:** `clear() → void`
- **Description:** Clears all thread-local state and the message resolver.

### Static Method
- **Signature:** `isI18nField(String value) → boolean`
- **Description:** Checks if a string value is an i18n key expression (wrapped in `${...}`).
- **Parameters:** `value` — the string to check
- **Returns:** `boolean`

### Static Method
- **Signature:** `i18nName(String value) → String`
- **Description:** Extracts the i18n key name from a wrapped expression. For `"${user.name}"`, returns `"user.name"`.
- **Parameters:** `value` — the wrapped expression
- **Returns:** `String` — the key, or null if not an i18n field

### Static Method
- **Signature:** `wrapKey(String key) → String`
- **Description:** Wraps a key string in the i18n prefix/suffix: `key → "${key}"`.
- **Parameters:** `key` — the raw key
- **Returns:** `String` — the wrapped expression

### Static Method
- **Signature:** `translate(Object value) → Object`
- **Description:** Recursively translates an object: `I18nObject` — lookup by current locale; `String` — resolve if it is an i18n key expression; `Map`/`List` — recursively translate entries; other types — returned unchanged.
- **Parameters:** `value` — the object to translate
- **Returns:** `Object` — the translated value

### Static Method
- **Signature:** `translateString(String value) → String`
- **Description:** Translates a single string value. If it is an i18n key (`${key}`), resolves it via the message resolver. Supports inline default values with the syntax `${key:default}` and locale-paired defaults separated by `|#|`.
- **Parameters:** `value` — the string to translate
- **Returns:** `String` — the translated string

### Static Method
- **Signature:** `isChineseLocale(Locale locale) → boolean`
- **Description:** Checks whether the given locale is a Chinese variant.
- **Parameters:** `locale` — the locale to check
- **Returns:** `boolean`

### Static Method
- **Signature:** `isEnglishLocale(Locale locale) → boolean`
- **Description:** Checks whether the given locale is an English variant.
- **Parameters:** `locale` — the locale to check
- **Returns:** `boolean`

## I18nMessageResolver

**Type:** `@FunctionalInterface`

Strategy for resolving an i18n key to a localized message string. Implementations may load from resource bundles, databases, or any other storage.

### Method
- **Signature:** `resolve(String key, Locale locale) → String`
- **Description:** Resolves an i18n key to a localized message.
- **Parameters:** `key` — the i18n key to resolve; `locale` — the target locale
- **Returns:** `String` — the localized message

---

## I18nObject

**Type:** class (extends `LinkedHashMap<String, String>`)

A locale-to-value map representing an internationalized string. Provides locale-aware lookup with fallback: exact match (e.g. `zh-CN`) → language-only (`zh`) → first available value.

### Static Method
- **Signature:** `of(String value) → I18nObject`
- **Description:** Creates an `I18nObject` with a single value for the default language (en).
- **Parameters:** `value` — the English value
- **Returns:** `I18nObject`

### Static Method
- **Signature:** `of(String locale, String value) → I18nObject`
- **Description:** Creates an `I18nObject` with a single locale-value pair.
- **Parameters:** `locale` — the locale key; `value` — the localized value
- **Returns:** `I18nObject`

### Static Method
- **Signature:** `of(Map<String, String> values) → I18nObject`
- **Description:** Creates an `I18nObject` from an existing locale-value map.
- **Parameters:** `values` — map of locale to value
- **Returns:** `I18nObject`

### Static Method
- **Signature:** `of(String... pairs) → I18nObject`
- **Description:** Creates an `I18nObject` from alternating locale-value pairs. Example: `I18nObject.of("en", "Hello", "zh", "你好")`.
- **Parameters:** `pairs` — alternating locale, value, locale, value, ...
- **Returns:** `I18nObject`
- **Throws:** `IllegalArgumentException` if the number of arguments is odd

### Method
- **Signature:** `defaultValue() → String`
- **Description:** Returns the value for the current thread locale.
- **Parameters:** none
- **Returns:** `String`

### Method
- **Signature:** `enValue() → String`
- **Description:** Returns the English (US) value.
- **Parameters:** none
- **Returns:** `String`

### Method
- **Signature:** `cnValue() → String`
- **Description:** Returns the Simplified Chinese value.
- **Parameters:** none
- **Returns:** `String`

### Method
- **Signature:** `value(Locale locale) → String`
- **Description:** Resolves the value for the given locale with fallback: exact match (e.g. `zh-CN`) → language-only (`zh`) → other locale variants in the same language group → first available value.
- **Parameters:** `locale` — the target locale
- **Returns:** `String` — the resolved localized value
