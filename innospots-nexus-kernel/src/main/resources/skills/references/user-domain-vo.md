# Package `user.domain.vo`

## UserProfileVO

**Type:** record

Read model for user profile data returned by user data operations.

### Components
- **Signature:** `userId, userName, displayName, realName, email, mobile, avatarKey, registerSource, status`
- **Description:** `userId` is the string primary key used by user persistence entities.
- **Description:** User identity, contact, registration source, and lifecycle status fields.
